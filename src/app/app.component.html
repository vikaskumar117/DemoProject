To Put Code on github

  1. Create new repository
  2. Open that local folder in cmd
  3. write git init
  4. git status
  5. git add .
  6. git status
  7. git commit -m "first commit" .
  8. git status
  9. git remote add origin https://github.com/vikaskumar117/DemoProject.git
  10. git push -u origin master
  11. Then put the username and password of github.


To Publish

  1. npm i -g angular-cli-ghpages
  2. ng b --prod --base-href "https://vikaskumar117.github.io/DemoProject"
